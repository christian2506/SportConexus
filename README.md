# SportConnexus

Facilitar al usuario las reservaciones de ambientes deportivos de manera presencial, mediante nuestra plataforma reducir 
los procesos manuales para aumentar los ingresos a largo plazo (reducir costos de los documentos en papel), Para ello el plan
de trabajo que emplearemos será, desarrollar una plataforma web responsive para la empresa que se encarga 
del servicio de “Rentar por horas ambientes deportivos”.
